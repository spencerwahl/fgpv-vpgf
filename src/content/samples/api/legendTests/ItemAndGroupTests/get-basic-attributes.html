LegendItem, LegendGroup and InfoSection Tests - Get Basic Attributes

Directory

LegendItem Tests:

Note: The following tests are for the Natural Gas Pipeline LegendItem

1. Should be 'legendNode'


2. Should be of form 'node_x' where x is one or more digits.


3. Should either be a sensible name for the layer or 'undefined'.



LegendGroup Tests:

Note: The following tests are for the Group with a Visibility Set LegendGroup

1. Should be 'legendGroup'


2. Should be of form 'group_x' where x is one or more digits.


3. Should either be a sensible name for the group or 'undefined'.


4. Should list names of the children in the group (or undefined for some)'.



InfoSection Tests:

Note: The following tests are for the so lazy can't move Cat Gif 🐱 InfoSection

1. Should be 'legendInfo'


2. Should be of form 'info_x' where x is one or more digits.


3. Should either be a sensible name for the info section or 'undefined'.
