Enumeration IdentifyMode

Identify mode can be set to none, one, or more options. If no options are provided, no part of the identify workflow is executed

Details

Details: = "details"

Display the identify results in the details panel. This option only works in conjunction with the Query option. Without Query, there will be no results to display in the details panel.

Haze

  • Defined in api/src/layers.ts:1176

Dehighlights all other layers and features except the identify results (if Highlight is set) or the marker (if Marker is set). The haze will not be applied if neitherMarkernorHighlight` is set.

Highlight

Highlight: = "highlight"
  • Defined in api/src/layers.ts:1170

Highlight the identify results on the map. If the Marker mode is set, highlighted features will replace the marker. Only works when Query is set.
