Adds steps to this instance of the Stepper service. a step object must have two properties: isActive and isCompleted; they are bound to the step-item template and determine the visual appearance of a step; isActive indicates that a step's content is exposed to the user and awaits user input; only one step should be active at a time; isCompleted indicates that user input satisfies this step's validation; step's number badge is replaced with a checkmark icon;
